// ARCHITECTURE

Tri-hybrid search.
All in SQLite.

Three search backends — one lightweight file. No external services, no port conflicts, no Docker overhead.

BM25 Keyword Search

SQLite FTS5 virtual table. Fast exact-match retrieval. Handles Vietnamese and special characters safely.

SQLite FTS5

Vector Semantic Search

sqlite-vec extension + FastEmbed ONNX (multilingual-e5-large, 1024-dim). Fully offline, no Ollama daemon required.

sqlite-vec + FastEmbed

Knowledge Graph

Pure Python BFS on SQLite tables. Entities, relationships, aliases. No FalkorDB, no Cypher. Just SQL.

SQLite + BFS

// FUSION

All 3 results fused via Reciprocal Rank Fusion (RRF) — best of keyword, semantic, and graph in every query.

// WORKFLOW

Agent-driven design.

Kioku Lite stores what you provide. No built-in LLM calls. The agent extracts entities from its own context — no extra API costs.

1

Save a memory

$ kioku-lite save \

"Met Alice at the Acme Corp office today" \

--mood happy --tags work,people

→ returns:

"content_hash": "a3f7b2..."

"hint": "Run kg-index to add KG entries"

2

Index entities (agent-driven)

$ kioku-lite kg-index a3f7b2 \

--entities '[

{"name":"Alice","type":"PERSON"},

{"name":"Acme Corp","type":"ORGANIZATION"}

]' \

--relationships '[

{"source":"Alice","target":"Acme Corp",

"rel_type":"WORKS_AT","weight":0.9}

]'

3

Search — tri-hybrid

$ kioku-lite search "Alice" \

--entities "Alice"

→ returns:

"results": [ ... ]

"graph_context": {

"nodes": [{"name":"Acme Corp",...}],

"evidence": [...]

}

4

Recall & Explain

# Graph traversal

$ kioku-lite recall "Alice" --hops 2

# Shortest path

$ kioku-lite connect "Alice" "Acme Corp"

# Timeline

$ kioku-lite timeline --from 2026-01-01

// QUICK START

Up and running in 2 minutes.

01 — INSTALL

$ pipx install "kioku-lite[cli]"

# or: uv tool install "kioku-lite[cli]"

02 — PRE-DOWNLOAD EMBEDDING MODEL (~1.1 GB, one-time)

$ kioku-lite setup

# Optional but recommended — model auto-downloads on first search if skipped.

03 — INTEGRATE WITH YOUR AGENT

$ kioku-lite init --global

# Installs SKILL.md → ~/.claude/skills/kioku-lite/

# Works with Claude Code, Cursor, Windsurf

$ kioku-lite install-profile mentor

# Set up a persona with built-in KG schema

04 — START REMEMBERING

$ kioku-lite save "First memory"

$ kioku-lite search "first"

$ kioku-lite timeline
