// ARCHITECTURE

Tri-hybrid search. All in SQLite.

Three search signals fused into one query. No external services, no Docker, no cloud LLM.

BM25

SQLite FTS5. Exact keyword match. Multilingual.

Vector

sqlite-vec + FastEmbed ONNX. 1024-dim embeddings. Fully offline.

Knowledge Graph

SQLite-backed graph. BFS traversal. Entity aliases. Open schema.

Results fused via Reciprocal Rank Fusion (RRF) then deduplicated.

Deep dive: System Architecture · Write Pipeline · Search Pipeline

// WORKFLOW

Agent-driven. No built-in LLM.

The agent saves text, extracts entities from its own conversation, then indexes them into the graph. Kioku Lite never calls an LLM.

// WRITE PHASE

$ kioku-lite save "Met Alice today" --mood happy

→ content_hash: a3f7b2...

$ kioku-lite kg-index a3f7b2 \

--entities '[{"name":"Alice","type":"PERSON"}]' \

--relationships '[{"source":"Alice","rel_type":"WORKS_AT","target":"Acme"}]'

// SEARCH PHASE

$ kioku-lite search "Alice" --entities "Alice"

→ BM25 + Vector + KG → RRF merged results

$ kioku-lite recall "Alice" --hops 2

$ kioku-lite connect "Alice" "Acme"

$ kioku-lite timeline --from 2026-01-01

// KNOWLEDGE GRAPH

Open schema. Custom personas.

Entity types and relationship types are plain strings, not fixed enums. Users can create custom personas by changing how their agent extracts entities.

Companion persona

Nodes: EMOTION, LIFE_EVENT

Edges: TRIGGERED_BY

Mentor persona

Nodes: DECISION, LESSON

Edges: LED_TO

// SETUP

Copy-paste. Your agent does the rest.

Each guide is a self-contained prompt. Paste it into your agent and it will install kioku-lite, set up identity files, and start remembering automatically.

// GENERAL AGENT

Claude Code / Cursor / Windsurf

Standard setup via kioku-lite init --global and install-profile. Works with any agent that supports SKILL.md.

// OPENCLAW

OpenClaw + Telegram

Setup for OpenClaw gateway agents. Derives SOUL.md and TOOLS.md from installed profile. Supports Telegram bots.

// QUICK INSTALL (3 commands)

$ pipx install "kioku-lite[cli]"

$ kioku-lite setup # download embedding model (~1.1GB, one-time)

$ kioku-lite init --global # integrate with your agent

// ROADMAP

Lite for personal. Full for enterprise.

kioku-lite is available now for personal use. Zero Docker, zero cloud dependency, single SQLite file.
kioku-full (dedicated graph DB + vector DB, multi-tenant, API keys) is under active development for team and enterprise deployments.
